## Ternary operation
**C#**
```
bool isDone = false;
string result = isDone ? "Success" : "Fail";
Console.WriteLine(result);
```
**Ts**
```
let isDone: Boolean = false;
let result: string = isDone ? "Success" : "Fail";
console.log(result);
```
**Js**
```
var isDone = false;
var result = isDone ? "Success" : "Fail";
console.log(result);
```
**Py**
```
isDone = False;
result = "Success" if isDone else "Fail";
print(result);
```
